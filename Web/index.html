Lets Start With the Whatsapp Export

To start with the game, you need to export your whatsapp chat. To do that, follow the steps below:

  1. Open the Whatsapp chat you want to export.
  2. Tap on the three dots in the top right corner.
  3. Select "More" from the dropdown menu.
  4. Tap on "Export chat".
  5. Choose whether to include media or not.(Optional)
  6. Save the exported file to your device.
